# Changelog

## 1.2.0 (2026-06-12)

- 中英双语（chrome.i18n，跟随浏览器语言）
- controller 地址支持 https，未写协议自动补全
- 默认 controller 地址改为 http://192.168.10.1:9090
- 更名为「Clash 指示器 / Clash Indicator」

## 1.1.0 (2026-06-12)

- 图标动态变色（绿直连 / 蓝代理 / 灰未知 / 红错误）
- 弹窗显示节点延迟、页面流量
- 页面资源去向汇总（webRequest 域名追踪 + 比例条）
- 弹窗内切换代理组节点
- 打开 Clash 面板入口

## 1.0.0 (2026-06-12)

- 首个版本：角标显示直连/代理，弹窗显示命中规则和节点链，controller 地址与 secret 可配置
