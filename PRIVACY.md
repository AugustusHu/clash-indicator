# 隐私政策 / Privacy Policy

**Clash 指示器 / Clash Indicator**（下称"本扩展"）

最后更新 / Last updated: 2026-06-12

## 中文

本扩展不收集、不存储、不传输任何用户数据。

- **网络请求观察**（webRequest 权限）：仅用于在浏览器本地记录当前标签页请求过的域名列表，以便与本地 Clash API 的连接记录匹配。该列表只存在于内存中，关闭标签页即清除，不写入磁盘，不发送至任何地方。
- **网络通信**：本扩展只与你在设置中填写的 Clash external controller 地址（通常是局域网或本机地址）通信，用于读取连接、规则、延迟信息和切换节点。不连接任何开发者服务器或第三方服务。
- **存储**：仅通过 Chrome 同步存储保存两项设置——controller 地址和 secret。
- **数据出售/共享**：无任何数据，故不存在出售或共享。

## English

This extension does not collect, store, or transmit any user data.

- **Web request observation** (webRequest permission): used solely to keep an in-memory list of domains requested by the current tab, so they can be matched against connection records from your local Clash API. The list never leaves the browser, is never written to disk, and is cleared when the tab closes.
- **Network communication**: the extension communicates only with the Clash external controller address you configure (typically a LAN or localhost address) to read connections, rules, and latency, and to switch proxy nodes. It never contacts the developer's servers or any third party.
- **Storage**: only two settings are saved via Chrome sync storage — the controller address and secret.
- **Sale/sharing of data**: no data exists to sell or share.
